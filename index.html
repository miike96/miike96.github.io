Tecnologías que utilizo

  • Java (17+): Lenguaje principal en mis proyectos, usado para backend, aplicaciones de escritorio y lógica empresarial.
  • Spring Boot: Framework robusto para construir APIs y aplicaciones web en Java. Lo utilizo con arquitectura por capas y controladores REST.
  • Maven: Gestor de dependencias y compilación, ideal para mantener proyectos escalables y estructurados.
  • JavaFX: Framework para interfaces gráficas en escritorio. Ideal para aplicaciones como gestores de tareas u organizadores.
  • Thymeleaf: Motor de plantillas para Java que permite renderizar HTML de forma dinámica y eficiente.
  • CSS3: Para el diseño visual personalizado de mis vistas HTML, incluyendo tablas y formularios.
  • SQLite / JDBC: Base de datos ligera utilizada en proyectos locales o de escritorio.
  • Git / GitHub: Control de versiones, trabajo en ramas y subida de proyectos profesionales a repositorios públicos.
