Tecnologías que utilizo

  • Java (17+): Lenguaje principal en mis proyectos, usado para backend, aplicaciones de escritorio y lógica empresarial.
  • Spring Boot 3: Framework robusto para construir APIs y aplicaciones web en Java con arquitectura por capas y controladores REST.
  • Spring Security 6: Autenticación y autorización basada en roles (RBAC), BCrypt y protección de endpoints.
  • Oracle XE / JPA / Hibernate: Base de datos relacional en Docker con ORM completo y consultas parametrizadas.
  • Maven: Gestor de dependencias y compilación para proyectos escalables y estructurados.
  • JavaFX: Framework para interfaces gráficas en escritorio.
  • Thymeleaf: Motor de plantillas para renderizar HTML de forma dinámica.
  • Bootstrap 5 / Chart.js / Leaflet: Frontend moderno con gráficos interactivos y mapas.
  • Docker: Contenedores para levantar servicios de base de datos en local.
  • Git / GitHub: Control de versiones y publicación de proyectos.
